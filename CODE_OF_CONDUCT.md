# Code of Conduct

Be kind. Assume good faith. Give the benefit of the doubt.

If you wouldn't say it to someone in person, don't say it in a review, an issue, or a PR comment.

If you see someone behaving in a way that violates this, open a [private security advisory](../../security/advisories/new) or file a confidential issue on GitHub. Maintainers will follow up privately.

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) in spirit. The summary above is the operational version.
